select current_database();

to list all databases: \l

connect ot different databae: \c <<databae_name>>;

to check current schema you are working in: select current_schema();

to list all the schemas in database: \dn

to change schema to 'xyz': set search_path to xyz;

to change user: set session authorization 'postgres';

to list tables/views in current schema: \d

to list only tables: \dt

to list only views: \dv

to discribe certain table: \d <<table_name>>

to list users, and their role: \du
